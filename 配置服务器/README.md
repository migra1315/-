## 使用规范

1. 考虑到系统存储空间分配，将代码和数据分开存放；

```
数据存储路径：/data/<USERNAME>（机械盘）
代码存储路径：/home/<USERNAME>/WorkSpace（固态盘）
```

2. 注意数据安全，请间隔一段时间手动备份一次；
3. 不用的中间数据/模型，尤其是在系统盘的内容，请及时删除
4. 请勿将密码等信息告知他人
